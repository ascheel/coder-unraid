All notable changes to this template will be documented in this file.

## [v2.27.7] - 2025-01-XX

### Added
- Initial release of Coder unRAID Community Application template
- Support for built-in database (quick start)
- Support for external PostgreSQL database (production)
- Docker socket access for workspace management
- Configurable environment variables via unRAID UI
